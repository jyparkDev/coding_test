# 두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.

+ ## 입력은 여러 개의 테스트 케이스로 이루어져 있다. 각 테스트 케이스는 한 줄로 이루어져 있으며, 각 줄에 A와 B가 주어진다. (0 < A, B < 10)
<br>

## 입력 예시
+ ## 1 1
+ ## 2 3
+ ## 3 4
+ ## 9 8
+ ## 5 2

```C
#include <stdio.h>
int main() {
	int a, b;
    while(scanf("%d %d",&a,&b) != EOF)
    {
        printf("%d\n",a+b);
    }
	
	
	return 0;
}
```
